<h3> 🛠 Este Proyecto fue desarrollado con las tecnologías como: PHP, PDO, MySQL, MVC, HTML5, CSS3 y Boostrap</h3>

### <h3> :computer: Este Proyecto cuenta con las siguientes funciones: </h3>

<p aligth="justify"> ⭐️ Función de Registro de Usuarios: Se realiza el registro de usuarios con los siguientes datos: nombre de usuario, correo electrónico, contraseña y repetir contraseña. </p>

<p aligth="justify"> ⭐️ Función de Inicio de Sesión: Se realiza el inicio de sesión mediante el ingreso del correo electrónico y contraseña ingresados en el registro. </p>

<p aligth="justify"> ⭐️ Función de Cierre de Sesión: Se destruye la sesión mediante la función de cerrar sesión. </p>

<p aligth="justify"> ⭐️ Función de Registrar electrodomesticos: Se registra cualquier electrodometicos con los siguientes datos: nombre, descripción, stock, precio y una imagen del electrodomestico. </p>

<p aligth="justify"> ⭐️ Función de editar electrodomesticos: Se puede editar la información ingresada por cada electrodomestico. </p>

<p aligth="justify"> ⭐️ Función de eliminar electrodomesticos: Se puede eliminar cualquier electrodomestico registrado en el sistema. </p>

<p aligth="justify"> ⭐️ Función de listar usuarios: Se puede listar todos los usuarios registrados en el sistema. </p>

<p aligth="justify"> ⭐️ Función de agregar información: Se puede ingresar información sobre las locaciones donde se puede agregar los electrodomesticos. </p>

<p aligth="justify"> ⭐️ Función de editar información de locaciones: Se puede editar información sobre las locaciones registradas. </p>

<p aligth="justify"> ⭐️ Función de eliminarinformación de locaciones: Se puede eliminar información sobre las locaciones registradas. </p>

<p aligth="justify"> ⭐️ Función de agregar marcas: Se puede agregar información sobre las marcas como: nombre, país, nombre de proveedor y contacto proveedor. </p>

<p aligth="justify"> ⭐️ Función de editar marcas: Se puede editar la información de todas las marcas registradas. </p>

<p aligth="justify"> ⭐️ Función de eliminar marcas: Se puede eliminar las marcas registradas. </p>

<p aligth="justify"> ⭐️ Función de Listar electrodomesticos: Se listan en tarjetas todos los artefactos electrodomesticos que se han regitrado en el sistema. </p>

### <h3> ⚡ Instalación del proyecto en su ordenador: </h3>
- Se debe ubicar en directorio donde se encuentre xampp y dirigirse hacia el directorio htdocs
- En el directorio htdocs puede clonar o descargar el repositorio para hacer funcionar el proyecto. 
- Se debe activar las siguientes opciones del panel de control de xampp: Apache y MySQL
- Una vez activadas las funciones de xampp se debe dirigir en el navegador a la siguiente dirección: http://localhost/phpmyadmin/
- Estando en el panel de phpmyadmin se procede a crear una base de datos con el nombre artefactos.
- El script para la base de datos se encuentra dentro del proyecto en el directorio db. 
- Una vez ya ingreado el script de la base de datos se procede a ir al navegador y poner lo siguiente: localhost/(nombre del proyecto) y con ello se ejecuta el proyecto.
